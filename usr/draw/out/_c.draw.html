<static> getOffset(canvas, orig, event, loc, offset) → {Vector}

Returns a vector specifying the offset of the given event from the given point on the given canvas.
Parameters:
Name Type Argument Description
canvas Object canvas on which offset is to be determined
orig Point point relative to which the offset is to be measured
event Object event whose position relative to the original point is to be determined
loc Point <optional>
 point that will be modified to show the location of the event relative to the canvas. Re-using this point will improve performance if the function is called repeatedly, such as on any mousemove event.
offset Vector <optional>
 optional vector in which to store the result
